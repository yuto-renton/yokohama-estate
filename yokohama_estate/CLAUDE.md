# 横浜中古マンション 相場定点観測システム

## このシステムの目的
横浜エリアの中古マンション相場を定期的に自動収集・分析し、
住宅購入判断に使える投資家目線のレポートを生成する。

## 現在のデータソース（2025-03-08 更新）
**国土交通省 不動産取引価格情報API**（SUUMO/HOMESからの変更済）
- API: `https://www.land.mlit.go.jp/webland/api/TradeListSearch`
- 認証不要・無料・公式の成約価格データ
- データは公表まで1〜2四半期遅延あり

## 検索条件（固定）
- 物件種別：中古マンション（Type = "中古マンション等"）
- エリア：横浜市全18区（コード 14101〜14118 を順次クエリ）
- 間取り：3LDK（全角対応済み：３ＬＤＫ も検出）
- 価格上限：6,000万円
- 取得期間：直近8四半期分

## 実行方法（これだけ覚えておけばOK）
```bash
cd ~/Downloads/yokohama_estate/yokohama_estate
python3 run.py
open reports/$(date +%Y-%m-%d).html
```

## ファイル構成
```
run.py                  ← メイン。これだけ実行すればOK
agents/mlit_agent.py    ← 国交省API収集エージェント（2025-03-08 新設）
agents/suumo_agent.py   ← 旧SUUMO用（現在未使用・ボット検出でブロックされる）
agents/homes_agent.py   ← 旧HOMES用（現在未使用・URLが404で使えない）
data/mlit_latest.json   ← 最新取得データ（run.py が自動生成）
data/history.csv        ← 週次で積み上がる相場履歴
reports/YYYY-MM-DD.html ← 日付付きHTMLレポート
```

## レポートに含まれる指標
- 平均成約価格・中央値・最安・最高
- 平均㎡単価（坪単価も併記）
- 前回比変動率（2回目以降）
- 価格帯別・築年別の件数分布
- 区別平均価格ランキング（安い順）
- 表面利回り試算（賃料20万/月想定）
