# 🏙 横浜中古マンション 相場定点観測システム

ゆーと専用。毎週の物件相場を自動収集・分析・レポート化する。  
不動産営業マンに言われるまま動かないために、自分でデータを持つ。

---

## ファイル構成

```
yokohama_estate/
├── CLAUDE.md              ← Claude Codeへの司令書（親エージェントの脳）
├── run.py                 ← 親エージェント（子2体を並列起動）
├── agents/
│   ├── suumo_agent.py     ← 子①：SUUMO担当
│   └── homes_agent.py     ← 子②：HOMES担当
├── data/
│   ├── history.csv        ← 週次で積み上がる相場データ（資産）
│   ├── suumo_latest.json  ← SUUMO最新取得データ
│   └── homes_latest.json  ← HOMES最新取得データ
├── reports/
│   └── YYYY-MM-DD.html    ← 週次レポート（ブラウザで見る）
├── setup.sh               ← 初回セットアップ（Mac）
└── README.md              ← これ
```

---

## セットアップ（初回のみ）

```bash
cd yokohama_estate
bash setup.sh
```

→ ライブラリインストール＋毎週月曜8時の自動実行を設定する

---

## 手動実行

```bash
python3 run.py
```

実行後、`open reports/$(date +%Y-%m-%d).html` でレポートを確認。

---

## Claude Code から使う場合

このディレクトリで `claude` を起動すると、  
CLAUDE.md を読んだClaude Codeが司令塔として動く。

```bash
cd yokohama_estate
claude
# → "今週の相場を収集して" と話しかけるだけでOK
```

Claude Codeは CLAUDE.md の条件（3LDK/6000万以内/徒歩5分）を  
自動で読み込んで、子エージェントを並列実行してくれる。

---

## レポートに含まれる指標

| 指標 | 説明 |
|------|------|
| 平均・中央値・最安・最高 | 基本統計 |
| 前週比変動率 | 相場トレンド確認 |
| ㎡単価・坪単価 | 物件の本当の価値比較 |
| 表面利回り試算 | 投資家目線の割安感チェック |
| 価格帯分布 | 市場の厚みを把握 |
| 区別ランキング | エリア選定の参考 |

---

## 検索条件（変更する場合）

`CLAUDE.md` の「検索条件」セクションを書き換えて、  
`claude` を起動すると反映される。
